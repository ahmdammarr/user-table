
## Getting Started

#### [https://users-table-smoky.vercel.app/](Demo)

run the development server:

```bash
yarn dev
```

run the test server:

```bash
yarn test
```

For deployment got to the [https://github.com/ahmdammarr/user-table](repo)

3- Click `Run wokflow` menu
